# Sample Images for Testing

Place sample images in this folder for testing:

- `profile.jpg` - Profile picture (any person photo)
- `medicine.jpg` - Medicine/tablet photo
- `document.pdf` - Sample medical document
- `license.jpg` - Sample license/certificate
- `ambulance.jpg` - Ambulance photo
- `lab-report.jpg` - Lab report sample

You can use any images from the internet or create placeholder images for testing.
